# Lo que aprendi hoy

¿Para qué sirve el comando COPY en el Dockerfile? : El comando COPY nos permite establecer desde donde se esta copiando a que se esta copiando desde el archivo app.y desde el proyecto local al contenedor que es /app con ayuda de la instruccion WORKDIR

¿Qué significa el mapeo -p 8080:8000?:
Este hace -p 8080:8000 un mapeo de puertos entre el host y el contenedor donde el 8080 es el del computador que se conecta con el 8000 que seria el del contenedor permitiendo el acceso desde el localhost:8080
